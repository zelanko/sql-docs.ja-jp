# <a name="data-types-in-odbc"></a>ODBC のデータ型
ODBC では、SQL データ型と C データ型の 2 つのデータ型セットを使用します。 SQL データ型はデータ ソースで使用され、C データ型はアプリケーションの C コードで使用されます。
