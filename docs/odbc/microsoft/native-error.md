# <a name="native-error"></a>ネイティブ エラー
正のネイティブ エラー コードは、デスクトップ データベース ドライバによって生成されます。 負のエラー コードは、マイクロソフトの Jet によって生成されます。 詳細については、『 Microsoft *Jet データベース エンジン プログラマ ガイド 』* を参照してください。
