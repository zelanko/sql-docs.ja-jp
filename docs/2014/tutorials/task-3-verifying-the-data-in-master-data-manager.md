1.  場合**マスター データ マネージャー**は既にを開き、 **SQL Server 2012 マスター データ サービス**上部のホーム ページに移動します。 それ以外の場合に移動します[ http://localhost/mds ](http://localhost/mds)を起動する**マスター データ マネージャー**します。  
  
2.  選択**Suppliers**の**モデル**、 をクリック**エクスプ ローラー**します。
