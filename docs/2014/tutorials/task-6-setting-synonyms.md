# <a name="task-6-setting-synonyms"></a>タスク 6: シノニムを設定する

1.  選択**国**ドメインの一覧から。  
  
2.  切り替えて、**ドメイン値**タブ。  
  
3.  クリックして**新しいドメインの値を追加**ツールバーのボタンをクリックします。  
  
4.  型**USA**値とキーを押して**ENTER**します。
