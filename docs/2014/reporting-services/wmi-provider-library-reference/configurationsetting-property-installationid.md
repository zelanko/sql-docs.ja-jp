# <a name="installationid-property-wmi-msreportserver_configurationsetting"></a>InstallationID プロパティ (WMI MSReportServer_ConfigurationSetting)
  このプロパティは、特定のレポート サーバー インスタンスの一意な識別子を返します。  
  
## <a name="syntax"></a>構文  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
