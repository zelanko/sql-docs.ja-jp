# <a name="type-property-column-adox"></a>Type プロパティ (列) (ADOX)
列のデータ型を示します。  
  
## <a name="settings-and-return-values"></a>設定と戻り値  
 設定または取得を**長い**のいずれかの値、[格納](../../../ado/reference/ado-api/datatypeenum.md)定数。 既定値は**adVarWChar**します。
