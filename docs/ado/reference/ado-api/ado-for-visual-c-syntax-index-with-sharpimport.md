 この特定のインデックスは、ADO .dll に対して **#import**ディレクティブを使用してプログラムをコンパイルし、 \*生成された tlh ファイルを再フォーマットすることによって得られたものです。 メソッド、プロパティ、およびイベントに関する情報のみが保持されました。 各プロパティに対して宣言された別の構文は`__declspec(property...)`、対応する "" ディレクティブによって示されます。  
  
 詳細については、 [VISUAL C++ ADO プログラミング](../../../ado/guide/appendixes/visual-c-ado-programming.md)を読むことを強くお勧めします。
